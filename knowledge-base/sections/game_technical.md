# Technical Operations with Game Servers

Changing game/core or reinstalling:
1. Go to the Server Panel → Settings section, CHANGE EGG block. Select the desired core and click Apply.
2. Enable the "CHANGE STARTUP COMMAND" and "REINSTALL SERVER" options. If you are changing the game or the server is completely new — enable "WIPE SERVER FILES" (warning: this will delete all files!). Click the Change Egg button.
3. Wait for the process to complete (about a minute); the server is ready to use.
Changing version (e.g., Minecraft 1.16.5): Go to Startup and specify the desired version. Then in Files delete all current files. After that, in Settings click REINSTALL SERVER — the core for the selected version will be downloaded automatically.
Free subdomains for Minecraft: *.20tps.monster or *.anti-ddos.site — Subdomains section.
Plugins: Plugins section (for supported cores). Mods: Mods section (for corresponding cores).
