# VDS — Operating Systems and Software

Available Linux OS: Ubuntu 24.04, 22.04, 20.04, 18.04; Debian 13, 12, 11, 10, 9; AlmaLinux 10.0, 9.x, 8.x; Rocky Linux 9, 8.4; CentOS 9 Stream, 8 Stream, 7; Fedora, OpenSUSE, Alpine Linux.
Windows: Server 2025, 2022, 2019 (Evaluation versions).
One-Click Apps: ISPmanager, Plesk (CentOS/Alma/Windows), cPanel (CloudLinux/Alma/Ubuntu), WordPress on Plesk, Cloudron, OpenVPN, OpenClaw, NextCloud, GitLab, Jenkins, MariaDB, NGINX, Jitsi Meet.
If the required software is not available — the client can install it manually after a clean OS installation. Do not offer to install it for the client.
