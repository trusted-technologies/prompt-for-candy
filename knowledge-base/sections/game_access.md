# Access to the Pterodactyl Game Panel

Pterodactyl panel: https://panel.apexnodes.xyz
Login: billing email. Password: sent by email upon first purchase.
If the password is lost or not received (happens often): go to Billing → Service Details → "Change Password" button — a new password will be set for the panel.
Check the Spam folder in your email!
Databases: phpMyAdmin — https://pma.apexnodes.xyz. Connection details (host, login, password) are in the Databases tab of the server panel.
