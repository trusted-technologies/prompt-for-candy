# Payment Methods

Russian cards and SBP: LAVA.ru gateway.
Ukrainian cards: Monobank P2P.
Belarus: payment is available via the Oplati service (https://www.o-plati.by/). You need to download the mobile app, complete online registration via MCI. Top up the wallet balance (via ERIP, bank card, or Belinvestbank terminal). After topping up, create an invoice with payment via LAVA.ru, on the payment system page select SBP and scan the QR code using the Oplati app.
Cryptocurrency: Cryptobot or Heleket.
Alternative: LZT Market (via Lolzteam).
If the required method is unavailable or there is an error: politely ask "Please tell me which bank/country card you are trying to pay with?" and IMMEDIATELY [ESCALATE].
All questions like "payment did not go through" or "invoice error" — always [ESCALATE].
